# Prophaze WAF

Prophaze Web Application Firewall (WAF) Helm chart for Kubernetes/OpenShift.

## Introduction
This chart deploys Prophaze WAF on a Kubernetes cluster using the Helm package manager.

## Prerequisites
- Kubernetes 1.20+
- Helm 3.0+

## Installing the Chart
```bash
helm install prophaze-waf ./prophaze-waf
```

## Configuration
See values.yaml for configurable parameters.
