Configuration notes:
- Set secrets.jwtSecret with your JWT signing secret
- Set config.database.host, config.database.source, config.database.username and secrets.databasePassword for the primary DB
- Set config.tracesStore.host and secrets.tracesStorePassword for ClickHouse
- Set config.metricsStore.url for VictoriaMetrics
