Your Expanso Edge Compute Node has been deployed.

Release name: {{ .Release.Name }}
Namespace: {{ .Release.Namespace }}

To check the status of your deployment:

  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "expanso-edge.name" . }}

To view the logs:

  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "expanso-edge.name" . }}

To check the API endpoint:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "expanso-edge.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.targetPort }}

Then access: http://localhost:{{ .Values.service.port }}/api/v1/agent/alive
